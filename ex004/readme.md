Nesse exercícios, estamos estudando como fazer favicons para nosso site, aquela pequena imagem que fica no cantinho do web-site, como Google, etc. 

Primeiramente, podemos já baixar a imagem pronta se for de alguma coisa específica, ou criarmos um ícone no favicon.cc, como podemos também transformar nossas imagens em ícone, conforme vamos ver a seguir. 

Lembrando que, para incluirmos o ícone em nosso site, devemos criar um link de referência dentro do nosso diretório, tem o site favicon.cc, e assim podemos também gerarmos um ícone através de uma imagem no favicon.io 

Após ter salvo a imagem de seu favicon, devemos colocar o seguinte endereço no código, logo acima do título: 

&lt;link="icon" href="images/m.png" type="image/png"&gt;

lembrando que, se usarmos no formato .ico, devemos colocar no campo "type="image/x-icon"", e assim sucessivamente.
