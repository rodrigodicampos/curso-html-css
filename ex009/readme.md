<b>Listas OL e UL</br>

Aqui, nesse módulo, iremos ver sobre listas OL e UL
O que isso significa? 
Listas ordenadas e não ordenadas. 

As listas ordenadas sao a tag OL 
e não ordenadas são as tags UL
Para listarmos os itens dentro da lista, iremos utilizarmos a tag li. 

Lembrando que a partir do HTML 5, o /li é opcional. 

Um parâmetro que eu posso usar:
<ol type="1">
<li>Acordar
<<li>Ligar para João
<li>Fazer jantar
</ol>

Mais detalhes estão dentro do proprio codigo. 

Em listas não ordenadas, ou seja, que não precisamos escrever números, podemos usar marcadores especiais, como quadrados, circulos, etc, conforme demonstrado dentro do nosso código. 

Podemos misturar as duas listas, tanto ordenadas quanto nāo ordenadas dentro de nosso código, como demonstrado a partir da linha 46. 

Colocar uma lista dentro da outra, é mais usado dentro de menus.

Listas de definições: para definir o que cada item, geralmente o item principal fica a direita, e a descrição do item mais a esquerda.
