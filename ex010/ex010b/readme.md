Para inserirmos links externos dentro de nosso site, devemos colocar no início do código &lt;a href=&gt;, e antes de fecharmos o código, devemos colocar o que queremos ancorar o link, por exemplo:

Para acessar ao site do Google, clique <a href="www.google.com.br" aqui>. Como pode perceber, o "aqui" fica ainda dentro do código. Mas, nessa marcação simples, o site irá abrir diretamente dentro de sua página, para isso, devemos incluir ainda dois detalhes dentro de nosso parâmetro: 

Exemplo: 
&lt;a href="https://www.google.com" target="_blank" rel="external"&gt;

<b>_blank</b>: O link irá ser aberto em uma nova aba.
